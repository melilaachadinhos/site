# Guia de dropshipping manual - meli.la achadinhos

Este guia organiza a operação sem automação de estoque e pedidos.

## Fluxo manual passo a passo

1. O cliente compra no seu site.
   O cliente acessa a página, escolhe o produto e preenche os dados de entrega no formulário.

2. Você recebe o dinheiro e os dados.
   Confira o pedido recebido no atendimento e copie o endereço de entrega informado pelo cliente.

3. Você faz o pedido no fornecedor.
   Acesse o AliExpress ou o site do fornecedor, adicione o produto ao carrinho e preencha o endereço de entrega com os dados do cliente.

4. Você paga o fornecedor.
   Pague o fornecedor usando o dinheiro que o cliente já pagou. O fornecedor envia o produto direto para a casa do cliente.

5. Você envia o rastreio.
   Alguns dias depois, quando o fornecedor postar o produto, copie o código de rastreamento e envie manualmente por e-mail ou WhatsApp para o cliente.

## Controles da pasta

- `controle-produtos-manual.csv`: use para cadastrar produto, SKU, preço, custo estimado, fornecedor e link.
- `controle-pedidos-manual.csv`: use para controlar cada pedido, atendimento, compra no fornecedor e rastreio.

## Status recomendados

- Novo pedido
- Aguardando confirmação
- Confirmado
- Comprado no fornecedor
- Enviado
- Entregue
- Troca ou suporte
- Cancelado

## Critérios para escolher fornecedor

- Produto com boas avaliações.
- Fotos e descrição confiáveis.
- Prazo de envio compatível com o que você promete.
- Preço com margem suficiente depois de frete, taxas e possíveis reembolsos.
- Fornecedor com histórico de pedidos.
- Produto que permita rastreio.

## Antes de vender

- Defina prazo realista de entrega.
- Escreva política de troca e reembolso.
- Defina como será feito o suporte.
- Teste um pedido com seu próprio endereço.
- Guarde comprovantes de compra no fornecedor.
- Nunca prometa entrega rápida se o fornecedor não garantir.

## Rotina diária

1. Verificar novos pedidos.
2. Conferir pedidos recebidos.
3. Comprar pedidos pagos no fornecedor.
4. Atualizar planilha de pedidos.
5. Enviar rastreios novos.
6. Responder clientes.
7. Conferir produtos com atraso.
